🚀 Intelligent Laravel Caching

Laravel SmartCache is a powerful caching optimization package designed to enhance the way your Laravel application handles data caching. It intelligently manages large data sets by compressing, chunking, or applying other optimization strategies to keep your application performant and efficient.

Perfect for Laravel applications looking to optimize cache performance, reduce memory usage, and improve application speed with intelligent data compression, modern SWR patterns, and comprehensive monitoring - all while maintaining the familiar Laravel API you love.

💻 Usage Examples

🚀 Quick Start

Get started instantly! SmartCache works exactly like Laravel's built-in Cache facade:

use SmartCache\Facades\SmartCache; // Basic caching (just like Laravel Cache) SmartCache::put('user_data', $userData, 3600); $userData = SmartCache::get('user_data'); // Helper function (just like cache() helper) smart_cache(['products' => $products], 3600); $products = smart_cache('products'); // Remember pattern (just like Cache::remember) $users = SmartCache::remember('users', 3600, function() { return User::all(); });

✨ The Magic: Your data is automatically optimized when beneficial!

⚡ Advanced Features

Enhance your applications with modern SWR patterns and smart invalidation:

// SWR: Serve stale data while refreshing in background $apiData = SmartCache::swr('github_repos', function() { return Http::get('api.github.com/user/repos')->json(); }, 300, 900); // 5min fresh, 15min stale // Pattern-based cache clearing SmartCache::flushPatterns([ 'user_*', // All user keys 'api_v2_*', // All API v2 cache '/product_\d+/' // Regex patterns ]); // Model auto-invalidation use SmartCache\Traits\CacheInvalidation; class User extends Model { use CacheInvalidation; // Cache automatically cleared when user changes! }

📊 Production Monitoring

Monitor & optimize your cache performance with real-time analytics:

// Real-time performance monitoring $metrics = SmartCache::getPerformanceMetrics(); $analysis = SmartCache::analyzePerformance(); // HTTP command execution (no SSH required) $commands = SmartCache::getAvailableCommands(); $result = SmartCache::executeCommand('status', ['force' => true]); // Monitoring dashboard example public function cacheHealthDashboard() { return response()->json([ 'performance' => SmartCache::getPerformanceMetrics(), 'health' => SmartCache::analyzePerformance(), 'commands' => SmartCache::getAvailableCommands() ]); }

🏢 Complex Applications

Enterprise-grade features for large-scale applications and systems:

// Complete e-commerce caching strategy class ProductService { // Fast-changing: SWR pattern public function getFeaturedProducts() { return SmartCache::swr('featured_products', fn() => Product::featured()->with('images')->get(), 300, 900 // 5min fresh, 15min stale ); } // Expensive: Refresh-ahead pattern public function getRecommendations($userId) { return SmartCache::refreshAhead( "recommendations_{$userId}", fn() => $this->aiEngine->generate($userId), 3600, 600 // 1hr TTL, 10min refresh ); } }

🚀 Quick Migration from Laravel Cache

SmartCache is 100% compatible with existing Laravel Cache code:

// Your existing Laravel Cache code works unchanged Cache::put('key', $value, 3600); $value = Cache::get('key'); // Just change the facade import // use Illuminate\Support\Facades\Cache; ❌ Old use SmartCache\Facades\SmartCache; // ✅ New // Now you get automatic optimization + new features SmartCache::put('key', $value, 3600); // Automatically optimized $value = SmartCache::get('key'); // Automatically restored // Plus new SWR methods are available immediately $value = SmartCache::swr('key', $callback); // 🆕 Modern pattern

Zero Breaking Changes - Safe upgrade for any Laravel application!
